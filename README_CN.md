# GitHubFollow
Gitstar自动加粉工具
# 快速入门
## Linux
### 第一步
安装Python 2.x,运行```python --version```and```pip```来确认安装成功。    
如果你已经安装过了，跳过这一步。  
__确认你已安装 ```requests```和```BeautifulSoup4```库__
### Step2
打开```main.py```,替换以下内容  
```
#-----------------------------
#GitStar
NAME="w568w"
PASSWORD="xxxx"
#GitHub
GITNAME="w568w"
GITPASSWORD="xxxx"
#-----------------------------
```
把它们改成你自己的信息。  
### Step3
运行```python -u main.py```,  
完事儿！
